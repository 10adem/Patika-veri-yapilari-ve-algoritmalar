# Proje 1

## Soru 1

[16,21,11,8,12,22]

Yukarıdaki dizinin sort türüne göre aşamalarını yazınız.

Big-O gösterimini yazınız.

```
[16,21,11,8,12,22]

1. [16,21,11]   [8,12,22]

2. [16] [21,11]   [8,12] [22]

3. [16] [21] [11]   [8] [12] [22]

4. [11,16,21]   [8,12,22]

Son olarak : [8,11,12,16,21,22]
```

Big-O :

```
O(nlogn)
```
